行为规划

今天又犯蠢了一次！

早上收到了搬瓦工续费邮件提醒，在没有经过仔细思考后就又完成了续费。当时续费的原有是这个 VPS 是现阶段最便宜的了不续费后续无法再购买的。当时抱着 IP 没有被封的侥幸心理完成了续费，但是结果就是 IP 依旧在防火墙列表内。真让人头大！

这次事件需要做如下几个总结教训：

  1. 对自己认识不充分：是否真的会心疼这 100 多块钱（事实是会心疼，如果 IP 被封无法正常用的情况下）
  2. 考虑问题不全面：IP 被封并且没有被移除的情况下，还需要继续购买其他翻墙工具，浪费更多的钱，在已有一个月备份 VPS 的情况下不应该这么不经思考就把 VPS 续费了；
  3. 对事情的决策犹豫不决：首先第一目标是实现翻墙，在已知搬瓦工无法更换 IP 的情况了为了所谓的便宜且无法再购买冲动消费，在临时 VPS 过期前有充足时间可以使用。临时 VPS过期后可以购买 dawang 软件，并且更便宜；

改进计划：

  1. 不论大钱小钱，如果没有产生价值，这个钱就是浪费了的，不能因为只是 几百块钱就冲动消费；
  2. 考虑问题需要做手动分析，需要考虑的几种场景，各个场景对应的后果及问题；
  3. 针对于 2 不断改正，对问题有一个正确的分析及认识后才会做到决策果断；

在公司或家庭中，遇事不决时找笔和纸进行分析后，在下结论。
