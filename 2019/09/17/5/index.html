Swift 真机运行报错：Reason：image not found

Swift 加载静态库 framework 时运行会报错，提示信息为：

分析原因：未正确加载 framework 文件；

解决方案：在 build Phases 选择 + 号，添加 new copy file phase ，选择 framework 类型，再添加相关 framework 文件
